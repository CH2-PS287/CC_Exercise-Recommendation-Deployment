# Food Recommendation Model Deployment with Flask

## Steps to Run

### 1. Create Virtual Environment

```
$ mkdir myproject
$ cd myproject
$ python3 -m venv .venv
```

### 2. Activate the Environment

```
. .venv/bin/activate
```

### 3. Install requirements/dependencies

```
pip install -r requirements.txt
```

### 3. Run Application

```
flask --app src run
```
